# 1. 项目概述

## 1.1 项目名称

云和气溶胶反演自动化计算系统（Cloud and Aerosol Retrieval Automation
System）

## 1.2 建设目标

本系统面向卫星遥感云参数和气溶胶参数反演任务，建设一个自动化、模块化、可扩展的遥感计算平台。

系统目标：

-   通过Web界面管理遥感反演任务
-   将算法封装为独立EXE模块
-   支持任务自动调度
-   支持HTCondor分布式计算
-   实现云反演和气溶胶反演流程自动化

------------------------------------------------------------------------

# 3. 当前支持功能

## 3.1 云反演模块

功能：

-   葵花8号卫星数据处理
-   云参数反演
-   云类型分析
-   云相态分析
-   CTH云顶高度反演

输入：

-   NetCDF卫星数据

输出：

-   云反演结果文件

调用方式：

    CM_CTH.exe config.json

------------------------------------------------------------------------

## 3.2 气溶胶反演模块

功能：

-   FY4A/FY4B卫星数据处理
-   气溶胶光学厚度AOD反演

输入：

-   FY系列卫星数据

输出：

-   AOD结果产品

------------------------------------------------------------------------

# 4. 系统运行流程

任务流程：

1.  用户创建任务
2.  选择反演模块
3.  指定输入数据
4.  系统生成任务
5.  调度系统分配资源
6.  调用EXE执行
7.  返回任务状态
8.  输出结果

# 6. 后端系统

目录：

    backend/

## main.py

FastAPI入口。

负责：

-   服务启动
-   API注册

## task_manager.py

任务核心管理。

负责：

-   创建任务
-   更新任务状态
-   管理任务生命周期

## module_manager.py

模块管理。

负责：

-   注册反演模块
-   管理EXE信息
-   保存模块配置

## htcondor_cluster_manager.py

HTCondor调度核心。

负责：

-   节点管理
-   资源检测
-   submit生成
-   任务提交
-   状态监控
